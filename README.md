# webpack2 + postcss + autoprefixer + bootstrap4

watch:

```
$ npm install
$ npm run start
$ open http://localhost:8080/
```

build:

```
$ npm run build
```

deploy:

deploy.json

```
{
  "host": HOST,
  "port": PORT,
  "username": USERNAME,
  "password": PASSWORD,
  "remoteRoot": REMOTE_ROOT
}
```

```
$ npm run deploy
```
